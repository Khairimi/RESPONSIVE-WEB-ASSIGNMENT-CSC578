JavaScript and Java

JavaScript and Java are similar in some ways but fundamentally different in some core structures.

The JavaScript language design resembles Java but has an option-typing system and lacks strict static typing rules.

typeof "Java" !== typeof "JavaScript";
Hello world

To get started with writing your first program, you can test it directly within your browser's dev tools console environment.

The standard syntax uses a console logging routine to push strings to the browser inspect panel.

console.log("Hello world");
Variables

You use variables as symbolic names for values in your application. The names of variables are called identifiers.

A JavaScript identifier must start with a letter, underscore, or dollar sign; subsequent characters can also be digits.

let variableName = 42;
